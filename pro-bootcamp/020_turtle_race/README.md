## Turtle race (day 20)

Ready for the challenge? Bet on one of the turtles in the game!

Run the race with:

``` 
$ python main.py
```

Enter your turtle color's, for example 'red'.

And best of luck!

Available colors are: 'red', 'blue', 'green', 'yellow', 'orange'.

Probably this doesn't capture all edge cases!
